FSPDS

View flipnotes in emulator with FSPDSNitro

2024-09-04 17:04:12 (3 month(s) ago)

The portability of the flipnote studio player FSPDS is contrained by the fact that it depends on an filesystem to scan flipnotes and store the app data. Due to this reason, FSPDS could only run on flashcards or modded hardware. However, one month ago I have released a tool called FSPDSNitro that allows embedding flipnotes into the ROM itself, making it possible to view flipnotes in emulators that support NitroFS. Check this link for more details.

FSPDS

FSPDS : Made it to 1.0

2023-08-07 11:29:00 (1 year(s), 4 month(s) ago)

More than two years and a half in development, the homebrew app that brings flipnotes to any DS console is no longer just a proof of concept. FSPDS has received a new user interface and an improved player, along with more useful features, including browsing the filesystem, playback options and viewing flipnote metadata such as the date and author names. My plans for the future of this application is creating an export option to different media formats and, if possible, a flipnote share/multicast feature via DS Download play.
