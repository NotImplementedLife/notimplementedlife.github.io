ObjAttribute * Astralbrew::Objects::OamPool::add_obj ( ObjAttribute  attr)

Adds new object to OAM.

Parameters
[in]attrnew object attributes
Returns
An identificator pointer of the object

Caution! Use the returned pointer attribute to work with the object. Do not attempt to free the pointer. Use remove_obj() to safely dispose of the object.

This function operates on a shadow memory zone. Use OamPool::deploy() to make the changes visible on the screen.
