If you only need to access what is at that address, use read() and write() methods instead.

◆ has_changed()

bool Astralbrew::Allocator::Address::has_changed ( )

Checks if the address has changed.

Returns
True if the address has changed, false otherwise

After this function call, the address "changed" flag is reset. Do not do this:

Address* address;
address->set_value(0x02000000); // change the address
if(condition1 && address->has_changed()) // has_changed() returns TRUE, changed flag reset
{ do_X(); }
if(condition2 && !address->has_changed()) // has_changed() returns FALSE!
{ do_Y(); }

Instead, do this:

bool address_changed = address->has_changed();
// good to go
if(condition1 && address_changed)
{ do_X(); }
if(condition2 && !address_changed)
{ do_Y(); }
