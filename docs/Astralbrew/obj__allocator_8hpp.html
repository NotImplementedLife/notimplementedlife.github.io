◆ alloc()

void Astralbrew::Allocator::ObjAllocator::alloc ( Address *  address,
int  blocks_magnitude 
)

Allocates a block of the specified size and exposes its address.

Parameters
[out]addressThe movable address containing the allocated OBJ VRAM pointer
[in]blocks_magnitudeSize of the allocated object block.

For the blocks magnitude, use the BLOCKS_SPR_<W>x<H>_<D>bit constants, where W, H are the object width, height (8,16,32,64), and D is the palette bit depth (4,8).

◆ dealloc()

void Astralbrew::Allocator::ObjAllocator::dealloc ( Address *  address)

Frees the OBJ VRAM block allocated at the address.

Parameters
[in,out]addressaddress of allocated block

If the deallocation succeeds, the value of the address is set to NULL. Freeing a previously unallocated or out of range address has no effect through this function call.
