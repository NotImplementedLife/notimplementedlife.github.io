Naxa NX-123

Description

Naxa NX-123 is a 2007 MP3 player which features audio and video playback, voice recording, FM radio, games, e-reader, picture viewing and others. It is a nice looking and portable device, however on this particular model the buttons are a bit hard to press, which becomes trivial after checking the inside glue tape which holds the buttons attached to the front shell. It's unclear if this is the result of a reparation or the original design.

Specifications

Screen 1.8in TFT
Memory 1GB
Battery 3.8V rechargable, 3.5 hours autonomy, max. 8 hours in power save mode
Port connectivty High Speed USB2.0, micro-B type
Sound playback MP1, MP2, MP3, WMA, WMV, ASF, WAV
Video playback AMV 128x128px
Voice recorder 8KHz WAV(32kbps), ACT(8kbps)
FM Radio Frequencies 76MHz–90MHz / 87MHz–108MHz, Max. stored channels 20+20, Max output (L)10mW + (R)10mW(32Ohm), SNR 45dB
Other e-Document reader, JPEG browser, Secret Space
